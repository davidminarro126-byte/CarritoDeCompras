# Práctica Refactorización: Carrito de Compras

Proyecto realizado para la asignatura de DAW.

## Cambios realizados durante la refactorización:
- **Separación de responsabilidades:** La lógica del carrito y la impresión de datos ahora están desacopladas.
- **Escalabilidad:** Se ha optimizado la estructura de datos para manejar grandes volúmenes de elementos.
- **Encapsulación:** Se han implementado Getters y Setters en `Item.java` con validaciones de seguridad.
- **Manejo de errores:** Se han añadido excepciones para precios inválidos y gestión de ítems inexistentes.
- **Documentación:** Se ha incluido Javadoc en el código fuente.
